# Programming Tails

The ultimate study platform for modern learners.

## Features
- AI Study Hub powered by Programming Tails Bot.
- Interactive Quizzes and Games.
- Global Leaderboard.
- Course and Blog management.

## Getting Started
### Web
1. `npm install`
2. `npm run dev`

### Mobile (Flutter)
1. `fvm flutter pub get`
2. `fvm flutter run`
